# tetris-game-console
Tác giả: Quản Lượng
IDE: CodeBlock
- Cách chơi: 
+ Phím A,D để di chuyển khối qua lại
+ Phím W để xoay khối
+ Phím S để khối di chuyển nhanh hơn
+ Phím X để thoát game
